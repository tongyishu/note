# OSPF

OSPF（Open Shortest Path First，开放式最短路径优先）

AS（Autonomous System，自治系统）

IR（Internal Router，内部路由器）

ABR（Area Border Router，区域边界路由器）

ASBR（Autonomous System Boundary Router，自治系统边界路由器）

AS-external（Autonomous System External，外部自治系统）

LSA（Link State Advertisement，链路状态通告）

LSDB（Link State Database，链路状态数据库）

Area: 区域

Area 0: 骨干区域

Area 1...N: 非骨干区域

```bash
[AS-External Network]  
       | (Type 5 LSA)  
       ↓  
     +------+  
     | ASBR | ← 外部路由注入（如BGP/静态路由）  
     +------+  
       | (Type 5 LSA)  
       ↓  
+------------------+  
|     Area 0       |  
|  (骨干区域)       |  
|  +-----+         |  
|  | ABR | ← 区域间路由汇总（Type 3 LSA）  
|  +-----+         |  
+------------------+  
       | (Type 3 LSA)  
       ↓  
+------------------+  
|     Area 1       |  
|  (非骨干区域)     |  
|  +-----+  +-----+|  
|  | IR  |  | IR  || ← 内部路由（Type 1/2 LSA）  
|  +-----+  +-----+|  
+------------------+  

```

注：每个ABR必须连接一个骨干区域（Area 0）和一个非骨干区域（如Area 1）


# OSPF Header

每个OSPF数据包都以一个标准的24字节报头开头。该报头包含了确定该数据包是否应被接收以便进行进一步处理所需的所有信息。

```bash
       +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
       |   Version     |     Type      |         Packet length         |
       +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
       |                          Router ID                            |
       +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
       |                           Area ID                             |
       +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
       |           Checksum            |             AuType            |
       +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
       |                       Authentication                          |
       +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
       |                       Authentication                          |
       +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
```

Type：


| Type | Description                |
| :----- | ---------------------------- |
| 1    | Hello                      |
| 2    | Database Description       |
| 3    | Link State Request         |
| 4    | Link State Update          |
| 5    | Link State Acknowledgments |

# OSPF LSA Header

```bash
       +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
       |            LS age             |    Options    |    LS type    |
       +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
       |                        Link State ID                          |
       +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
       |                     Advertising Router                        |
       +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
       |                     LS sequence number                        |
       +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
       |         LS checksum           |             length            |
       +-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+
```

所有LSA都以一个通用的20字节报头开始。这个报头包含了足够的信息来唯一地识别该LSA（LSA类型、链路状态标识符以及通告路由器）。

LS Type：


| LS Type | Description         |
| :-------- | --------------------- |
| 1       | Router-LSAs         |
| 2       | Network-LSAs        |
| 3       | Summary-LSAs (ABR)  |
| 4       | Summary-LSAs (ASBR) |
| 5       | AS-external-LSAs    |
